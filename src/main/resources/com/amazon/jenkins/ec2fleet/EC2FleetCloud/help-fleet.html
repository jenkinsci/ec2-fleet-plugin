Select the fleet which will be used to launch Jenkins workers. EC2 Spot Fleet and Auto Scaling Group are both supported.

By default, this list only includes active EC2 Spot Fleets and Auto Scaling Groups. If you need to see all fleets, click the checkbox below. The selected fleet will be in the list regardless of fleet state.

EC2 Spot Fleet specific: All non maintain type fleets will be filtered out from this list. More about types. If you are not sure about your fleet type, check its type via an API call or look in the console UI for Maintain Target Capacity being enabled.

AWS Permissions: The plugin requires certain AWS permissions for managing resources. You can check missing permissions by clicking the Test Connection button. The Test Connection button does not validate certain listed permissions such as TerminateInstances and UpdateAutoScalingGroup due to API restrictions. Check out the README for an inline IAM Policy.
